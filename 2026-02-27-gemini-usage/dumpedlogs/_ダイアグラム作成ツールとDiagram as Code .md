# you asked

こういうダイアグラムってどんなツールで書く？Diagram as Code がある？

---

# gemini response

ご提示のダイアグラムのような手書き風のスタイルで描画できるツールは複数あります。

## 描画ツール

* **Excalidraw**: 手書き風のスタイルが特徴のオープンソースの描画ツールです。ブラウザで簡単に利用でき、チームでの共同作業も可能です。ご提示の画像に最も近いスタイルです。
* **Miro**: オンラインホワイトボードツールで、豊富なテンプレートと手書き風の描画機能も備えています。ブレインストーミングやワークフローの可視化に適しています。
* **Figma**: UI/UXデザインツールとして有名ですが、ベクターグラフィック機能を使ってダイアグラムを作成することも可能です。プラグインで手書き風のスタイルを適用することもできます。

---

## Diagram as Code

「Diagram as Code」の考え方に基づき、コードでダイアグラムを生成するツールも存在します。これらはバージョン管理システムとの相性が良く、コードレビューがしやすいのが利点です。

* **Mermaid**: Markdownのようなシンプルな記法でフローチャートやシーケンス図などを生成できます。GitHubやGitLabなど多くのプラットフォームでネイティブにサポートされています。
* **PlantUML**: テキストベースの記法で、UML図を中心に様々な種類のダイアグラムを作成できます。非常に多機能で、複雑なダイアグラムも記述可能です。
* **Draw.io (diagrams.net)**: Webベースの描画ツールですが、XML形式でダイアグラムのデータを保存でき、テキストとしてバージョン管理が可能です。

これらのツールは、コードベースでダイアグラムを管理したい場合に便利です。特にMermaidやPlantUMLは、ドキュメントに直接ダイアグラムのコードを埋め込むことができ、メンテナンス性を高めるのに役立ちます。
