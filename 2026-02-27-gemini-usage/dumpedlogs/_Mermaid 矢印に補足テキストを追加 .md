# you asked

graph LR
  SI --> HRM
  HRM --> PA

Mermaid 図中の SI → HRM の矢印に補足を書きたいんだけど、どうする？

---

# gemini response

Mermaidで矢印（リンク）にテキストを追加するには、主に **パイプ `|` を使う方法** と **ハイフン `--` を使う方法** の2通りがあります。

ご希望の `SI --> HRM` の箇所に補足を入れる修正案です。

### 方法1：パイプ `|` を使う（一般的で簡単）

矢印の直後に `|テキスト|` を挟みます。

```mermaid
graph LR
  SI -->|補足テキスト| HRM
  HRM --> PA
```

**コード:**

```mermaid
graph LR
  SI -->|補足テキスト| HRM
  HRM --> PA
```

-----

### 方法2：ハイフン `--` を使う

矢印の線を伸ばして、間にテキストを挟みます。見た目は同じですが、コード上でテキストが埋もれにくい書き方です。

```mermaid
graph LR
  SI -- 補足テキスト --> HRM
  HRM --> PA
```

**コード:**

```mermaid
graph LR
  SI -- 補足テキスト --> HRM
  HRM --> PA
```

### ヒント

  * **改行したい場合:** テキスト内で `<br>` を使うと改行できます。（例: `|補足<br>テキスト|`）

-----
