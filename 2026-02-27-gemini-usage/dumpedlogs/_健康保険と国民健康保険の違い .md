# you asked

国保組合は「健康保険」ではない？健康保険と呼ばれるのは「健保組合」と「協会けんぽ」だけ？

国保組合、市町村国保、健保組合、協会けんぽをすべて包括した概念の名前は？

---

# gemini response

ご質問ありがとうございます。保険の制度は名称が似ていて非常に紛らわしいですよね。

結論から申し上げますと、**ご認識の通りです。**

厳密な法律上の定義では、**国保組合は「健康保険」ではありません。**
また、これらすべてを包括する名称は\*\*「公的医療保険」\*\*（または単に「医療保険」）となります。

以下に、頭の中をすっきり整理できるよう詳しく解説します。

-----

### 1\. 「健康保険」と「国民健康保険」の明確な違い

法律（根拠となる法律）が違うため、使い分けられています。

  * **健康保険（被用者保険）**

      * **根拠法：** 健康保険法
      * **対象：** 会社員、教職員、公務員など「雇われている人」
      * **ここに含まれるもの：**
          * **組合健保**（大企業の健保組合）
          * **協会けんぽ**（中小企業の従業員）
          * ※広義には共済組合（公務員等）も含んで語られることが多いです。

  * **国民健康保険（国保）**

      * **根拠法：** 国民健康保険法
      * **対象：** 自営業、フリーランス、無職の方など
      * **ここに含まれるもの：**
          * **市町村国保**（一般的な国保）
          * **国保組合**（医師、建設業、理容師など同業種の集まり）

したがって、**「国保組合」はあくまで「国民健康保険」の一種**であり、狭義の「健康保険」とは区別されます。

### 2\. すべてを包括する名前は？

国保組合、市町村国保、健保組合、協会けんぽ（さらに共済組合や後期高齢者医療制度まで）をすべてまとめた概念の名前は以下の通りです。

> **公的医療保険（こうてきいりょうほけん）**

日常会話では単に「医療保険」や「保険証の種類」と呼ばれることもありますが、制度全体を指す正しい用語は「公的医療保険制度」です。

### 3\. 全体像の整理（分類表）

これら4つの立ち位置を表にまとめると、以下のようになります。

| 包括的な名称 | 大分類（通称） | 具体的な運営主体（保険者） | 対象者のイメージ |
| :--- | :--- | :--- | :--- |
| **公的医療保険** | **健康保険**<br>（被用者保険） | **① 組合健保** | 大企業の会社員とその扶養家族 |
| | | **② 協会けんぽ** | 中小企業の会社員とその扶養家族 |
| | **国民健康保険**<br>（地域保険） | **③ 市町村国保** | 自営業、無職、退職者など |
| | | **④ 国保組合** | 医師、土木建築、理容師、芸能関係など<br>特定の業種の従事者 |

### 補足：なぜ「社会保険」という言葉と混ざるのか？

よく求人票などで「社会保険完備」と書かれていますが、この場合の「社会保険」は一般的に **「①組合健保 または ②協会けんぽ ＋ 厚生年金」** のセットを指します。

国保組合や市町村国保加入者は、年金については「国民年金」となるため、いわゆる狭義の「社会保険（社保）」の枠組みとは区別して扱われるのが一般的です。

      * はい、法律上は「国民健康保険」のグループです。
  * **「健康保険」と呼ばれるのは「健保組合」と「協会けんぽ」だけ？**
      * はい、厳密にはその通りです（共済組合も似た性質ですが別法律です）。
  * **すべてを包括する名前は？**
      * **「公的医療保険」** です。
