# you asked

数値を入力する UI で、上下向きの三角形で 1 ずつ値を選ぶ UI のことを何と言う？

---

# gemini response

そのUIは一般的に「**スピンボックス**」（Spin Box）と呼ばれます。

---
## 他の呼び方

文脈やプラットフォームによって、以下のように呼ばれることもあります。

* **スピナー** (Spinner)
* **ステッパー** (Stepper)
* **ニューメリック・アップダウン** (Numeric Up/Down) - (特に .NET などの開発環境で)


これらはすべて、数値を入力するテキストボックスの横に、値を増やしたり（上向き▲）減らしたり（下向き▼）するための小さなボタンが配置されたコントロールを指します。
