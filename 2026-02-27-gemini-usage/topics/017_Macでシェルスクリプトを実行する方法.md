# you asked

Mac で shellscript を実行するには？

---

# gemini response

Mac でシェルスクリプトを実行するには、いくつかの簡単なステップを踏むだけです。ターミナルアプリを使います。

### 1\. ターミナルを開く

まず、Mac の **「アプリケーション」** フォルダ内の **「ユーティリティ」** フォルダにある **「ターミナル」** アプリを開きます。Spotlight 検索（`Cmd` + `Space`）で「ターミナル」と入力して開くのが最も速い方法です。

### 2\. シェルスクリプトを作成する

シンプルなシェルスクリプトを例として作成します。テキストエディタで以下の内容を記述し、デスクトップに `hello.sh` という名前で保存してください。

```sh
#!/bin/bash
echo "Hello, Shellscript!"
```

  * **`#!` (シバン)**: スクリプトの1行目にある `#!/bin/bash` は、このスクリプトをどのシェルで実行するかを指定しています。これは非常に重要です。

### 3\. 実行権限を付与する

作成したスクリプトファイルに実行権限を与える必要があります。ターミナルで以下のコマンドを実行します。

```sh
chmod +x ~/Desktop/hello.sh
```

  * **`chmod`**: ファイルのパーミッション（権限）を変更するコマンドです。
  * **`+x`**: 「実行権限を追加する」ことを意味します。

### 4\. スクリプトを実行する

実行権限を付与したら、以下のコマンドでスクリプトを実行できます。

```sh
./Desktop/hello.sh
```

  * **`./`**: これは「カレントディレクトリ（現在いるディレクトリ）にある」という意味です。ターミナルがどのディレクトリにいても、絶対パス（例: `/Users/yourname/Desktop/hello.sh`）か、相対パス（例: `~/Desktop/hello.sh`）で実行できます。

コマンドを実行すると、ターミナルに「Hello, Shellscript\!」と表示されます。

-----

### その他の実行方法

実行権限を付与せずにスクリプトを実行することも可能です。その場合、スクリプトを直接シェルに読み込ませて実行します。

```sh
bash ~/Desktop/hello.sh
```

この方法では `#!/bin/bash` の指定が無視され、`bash` コマンドでスクリプトが実行されます。スクリプトが単純なものであればこの方法でも問題ありませんが、一般的には**実行権限を付与して `./` で実行する方法**が推奨されます。
