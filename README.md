# Tau's Playground - PdM Sandbox and Idea Scraps

## 💡 Overview
This repository serves as a scrapbook for **technical verifications, UI mockups, temporary scripts, and idea fragments** arising from my work as a PdM.

It contains archived experimental code (PoC) generated during daily tasks and is not tied to any specific product.

## ⚠️ Disclaimer
- **Not Production Ready**: All code contained here is experimental and is not intended for use in production environments.
- **No Warranty**: Security, performance, and maintainability have not been taken into consideration.
- **Copy-Paste Caution**: Please do not copy and paste implementation logic directly into product code.

## 📁 Directory Structure
Each project or experiment is managed in a flat structure using dated folders.

```text
root/
 ├── YYYY-MM-DD-feature-idea-A/    # Mockup for a specific feature
 ├── YYYY-MM-DD-api-test-script/   # Disposable code for API verification
 └── ...
 ```
